A Estrutura do HTML

Toda página HTML começa com uma estrutura básica, que inclui a declaração do tipo de documento, a tag <html>, a seção <head> com informações da página, e a seção <body>, onde fica o conteúdo visível ao usuário.

As Principais Tags

As tags HTML definem o conteúdo da página. Algumas das mais utilizadas atualmente são:

  • <h1> até <h6>: usadas para definir títulos e subtítulos.
  • <p>: define um parágrafo de texto.
  • <a href="">: cria links para outras páginas ou sites.
  • <img src="">: insere imagens na página.
  • <ul>, <ol> e <li>: criam listas não ordenadas, ordenadas e seus itens.
  • <input> e <form>: utilizados em formulários.
  • <button>: cria botões interativos.

Trabalhando Com Textos

O HTML permite organizar e destacar textos com diversas tags e atributos úteis, como:

  • <strong>: deixa o texto em negrito.
  • <i>: coloca o texto em itálico.
  • <mark>: destaca um trecho do texto.
  • <u>: sublinha o conteúdo.
  • <br>: insere uma quebra de linha.
  • <blockquote>: usado para citações longas.
  • style (atributo): usado em qualquer tag para aplicar estilos CSS diretamente, ex: <p style="font-size: 18px;">
